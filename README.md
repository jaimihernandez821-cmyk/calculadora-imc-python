# Calculadora de IMC - Proyecto Bootcamp

## Desarrollo
Para este programa utilicé **Python** y apliqué conceptos de:
* **Casting:** Conversión de `input()` (strings) a `float` e `int`.
* **Manejo de Excepciones:** Bloque `try-except` para validar que el usuario ingrese números.
* **F-strings:** Para imprimir resultados con formato profesional de dos decimales.

## Reflexión
Este proyecto me ayudó a entender que el código debe ser robusto. No basta con que calcule bien; debe guiar al usuario si comete un error al ingresar sus datos. Aprender a manejar errores con `try-except` fue mi parte favorita del proceso.
